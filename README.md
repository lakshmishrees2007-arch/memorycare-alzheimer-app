# 🧠 MemoryCare – Alzheimer's Detection & Monitoring System

MemoryCare is a web-based healthcare application designed to assist in the early detection, monitoring, and management of Alzheimer's disease. The platform provides tools for patients and caretakers to track cognitive health, monitor symptoms, manage medications, and improve daily care routines.

## 📌 Features

### 👤 Patient Module
- Patient registration and profile management
- Alzheimer's risk assessment dashboard
- Facial expression analysis
- Speech pattern analysis
- Memory game for cognitive testing
- Symptom questionnaire and scoring
- Heart rate monitoring simulation
- Medication reminders
- Daily task management
- Personalized care plan

### 👨‍⚕️ Caretaker Module
- Caretaker dashboard
- Patient monitoring
- Live alerts and notifications
- Medication management
- Task planning and tracking
- Patient health overview
- Care plan management

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

The primary objective of MemoryCare is to assist in:
- Early detection of Alzheimer's symptoms
- Continuous patient monitoring

- Enhancing patient quality of life

## 🔒 Data Storage

Currently, data is stored locally using the browser's Local Storage. No external database or cloud service is required.
